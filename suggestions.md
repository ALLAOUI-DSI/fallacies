# Suggestions to improve Fallacy Fighter

- Add streak-based rewards (daily streak, weekly streak, and milestone badges) to increase retention.
- Introduce timed challenge mode with optional countdown pressure for advanced players.
- Add “Explain in your own words” free-text reflection after each quiz question.
- Offer adaptive question selection that increases/decreases difficulty from recent performance.
- Add per-category mastery tracking (relevance, causality, ambiguity, etc.).
- Include an onboarding mini-tutorial with two guided quiz examples for new users.
- Provide accessibility options: dyslexia-friendly font, larger default text, and reduced motion.
- Add export/import for local progress backup.
- Add optional multiplayer “debate duel” mode where players identify fallacies in the same prompt.
- Create teacher/classroom mode with assignable level packs and progress snapshots.
